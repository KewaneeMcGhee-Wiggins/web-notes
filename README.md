# Web Notes

A small Express + EJS app for jotting down class notes: it lists existing notes and lets you add a new one through a form, which posts back to the server and re-renders the list.

## Tech stack

- Node.js, Express, EJS, body-parser
- Bootstrap (via CDN) for styling

## Running it

```bash
npm install
node app.js
```

Then open `http://localhost:3000`.
